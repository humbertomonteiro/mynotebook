Títulos, paragrafos e algumas dicas:

A tag <h1> é um título de grau 1, existem 6 niveis de títulos. Niveis de títulos tem sua importância, não se usa como forma de tamanho de texto, pois isso é feito no 'CSS' onde vamos fazer o design do site. Então você não vai usar a tag <h3> porque acha que o tamanho é melhor. São niveis para serem usados como títulos e subtítulos e na hora de trabalhar com seu arquivo, ser bem mais pratico e profissional.

A tag <p> cria um paragrafo.(São os paragrafos que você está lendo) Você pode escrever o paragrafo de qualquer jeito, basta colocar no meio do par de tags <p> </p>.

Se precisar querbrar o texto em algum lugar específico, você pode usar a tag <br>.

Para colocar o sinal de maior (<) ou menor (>) no HTML, basta colocar assim '& l t ;' (Tudo junto) para o maior e'& g t ;' (Tudo junto) para menor.

A tag <hr> cria uma linha horizontal.(Igual a linha acima)

A tag <br> quebra a linha do texto.(Pois se der enter não vai fazer nenhuma diferença no texto)

Dica: Envelopar

Uma dica para deixar um texto entre tags é usando o envelopamento. Selecione o texto no seu 'Visual Studio Code',em seguida no seu teclado aperte 'ctrl' 'shift' 'p', aparecerá uma caixa de pesquisa e lá você busca 'abb'. Vão aparecer algumas opções e você irá escolher a opção 'Wrap with Abbreviation' em seguida vai buscar 'p' e dar enter. Agora toda vez que precisar colocar algum texto entre tags é só selecionar o texto, usar esse atalho:'ctrl' 'shift' 'p', dar 'enter'e escrever a tag desejada e dar 'enter' novamente.

Principais formatações de texto:

Como usar Negrito e Itálico do jeito certo:

Negrito se usa para dar destaque. Para fazer um texto ficar em negrito temos a tag <strong>. ex: Essa frase está entre tags <strong> </strong>.

Itálico se usa para dar ênfase. Para fazer um texto ficar em Itálico tem a tag <em>. ex: Essa frase está entre tags <em> </em>.

Texto grande e texto pequeno:

Estamos fazendo um texto grande e um texto pequeno nesse parágrafo. Usando a tag <big> para grande e a tag <small> para textos pequenos.

Texto deletado:

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. O texto vai ficar riscado, usando a tag <del>. Ex: Estou escrevendo tanto que estou até impressionado.

Mais formatações:

Temos a tag <code> que deixa o texto com uma formatação mais fácil de se ler. São usadas geralmente para mostrar códigos de progamação

Ex sem tag: document.getElementById('teste') Ex com tag: document. getElementById('teste').

Chamamos esse essa fonte de mono espaçal. Mas usando apenas a tag <code> não vai dar para escrever código com a formatacão correta. Como exemplo olhe esse código em python:

num = int(input('Digite um número:')) if num % 2 == 0: print(f'O número {num} é PAR') else: pront(f'O número {num} é IMPAR') print('Fim do programa')

Citações simples:

Temos a tag <q> para colocar uma citação entre aspas, dá para colocar diretamente as aspas na hora de escrever,mas o HTML5 é usado para dar sentido ao texto e não simplesmente uma forma bonita ou interativa.(isso é papel do CSS e do jávaScript).

Citações completas:

Temos a tag <blockquote>, que servem para citaçoes maiores e vai deixar o texto em bloco.

Esse tipo de citação pode usar um parêmetro para dizer a fonte da citação dando o devido crédito ao autor. basta na tag de abertura escrever: cite="e aqui colocar a url de onde você tirou a citação ou se foi de um livro, colocar o link para comprar-lo"

Ex:

Abreviações:

Para abreviações temos a tag <abbr> quando colocar o mouse em cima da abreviação, vai mostrar o que você escrever na tag. Dentro da tag de abertura você vai escrever: title= "Aqui vai colocar o significado da abreviação"

Ex: Estou aprendendo HTML e CSS. Estou adorando!

Símbolos especiais:

Como adicionar:

Muito simples, sempre começa com o 'e comercial' (Que é esse símbolo - &) e no final usa ' ; ' (Ponto e virgula).

Símbolos frequentes:

® - Marca registrada - '& reg' tudo junto e com ' ; ' no final. © - Copyrigth - '& copy' tudo junto e com ' ; ' no final. ™ - Trade Marquue - '& trade' tudo junto e com ' ; ' no final. Δ - Delta - '& Delta' tudo junto e com ' ; ' no final. ⇑ - Pra cima - '& uArr' tudo junto e com ' ; ' no final.

Moedas:

€ - Euro - '& euro' tudo junto e com ' ; ' no final. £ - Libra - '& pound' tudo junto e com ' ; ' no final. ¥ - Yen - '& yen' tudo junto e com ' ; ' no final. ¢ - Centavo Americano - '& cent' tudo junto e com ' ; ' no final.

Mais símbolos no site da erikasarti, tem uma variedade muito grande.
