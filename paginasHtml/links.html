Links em HTML:

Tipos de links

Links externos:

Você pode usar links usando a tag <a>. Vai aparecer a referência, da seguinte forma:

<a href="e aqui você coloca o link desejado". É recomendado que depois da referência usar target="_blank" rel="external" pois quando clicar no link, o mesmo irá se sobrepor ao seu, então o primeiro parâmetro vai fazer abrir o link numa outra aba do navegar. Caso não use quem está navegando no seu site, quando clicar no link dificilmente valtará para o seu site. O outro parâmetro "rel" é usado como uma referência para o navegador, indica como você quiser para onde deve ser o próximo passo a ser seguido. O navegador pade dar a "rel" caso não coloque, então é muito importante o uso desse parâmetro.

Usando links internos:

Você pode criar uma outra págia no seus arquivos segunda página é só abrir um novo arquivo HTML no 'Visual Studio Code'. Igual como os links externos. Basta usar a tag <a> e colocar a url do seu novo arquivo. Aqui você pode apenas usar o ctrl espaço para buscar sua página criada. Caso esteja na mesma pagina achará facilmente, mas se estiver dentro de outra pasta terá que usar o / para acessar a pasta raiz da aplicação ou ../ para sair de uma pasta. Se quiser sair de mais pastas, é só continuar colocando ../ até achar a pasta desejada. NO caso desse exemplo foi foito assim:

<a href="../pg2.html" rel="next">Página 2.>

Também pode criar um novo arquivo em uma pasta diferente, mas não poderá simplesmente colocar o nome do arquivorecém criado, como no exemplo anterior, pois dará erro. Para resolver esse erro basta usar o atalho 'ctrl' 'espaço' que vão aparecer todos os arquivos e pastas que você criou. Busque o arquivo abrindo as pastas e selecione seu link para funcionar.

Para entender melhor, se fizermos uma pasta com o nome 'maispg' e criarmos a 'pg3.html'. Vamos ter qeu buscar assim: 

<a href="maispg/pag003.html">Páginas.>
Ficará com o nome da pasta antes do nome do arquivo, mostrando o caminho para achar o arquivo solicitado.

links Ancoras:

Podemos usar uma classe ou id para usarmos como referencia nos links e fazer com que levemos o usuario da página para partes especificas do arquivo.

Criando tags já com class ou id:

É bastante usual o uso de class e id para tags, usamos em css e ate em links de ancora. para criar sua tag ja com a class basta que você escreva a tag toda e use o . e o nome da class que dará a tag. Com o id é da mesma forma, mas trocaremos o . por #.

Adicionei um id para o primeiro assunto de links dessa página e referenciei o próximo link com esse id, da seguinte forma:

<a href="#int">links Internos.>

Então quando clicar nesse link, você será ancorado para o id referenciado. Links Internos.

Links para download:

Fazer links para dawnload de PDF é muito simples, usa-se a tag <a> Também, mas possa ser que seu navegador não baixe seu arquivo e apenas o mostre, para baixar você pode clicar no segundo botão do mouse e baixar link. Pode Também tentar usar o parâmetro download com o nome do arquivo que quer baixar, se não funcionar tem mais uma configuração, o parâmetro type que usa application/pdf para mídia em pdf.

Vou exemplificar com o PDF do curso em video para download.

Para saber qual configuração usar em cada media type basta buscar no site iana o tipo de mídia que irá usar. CLica no link, então no teclado aperta ctrl f vai aparecer uma barra de busca e é só buscar o tipo de arquivo que está usando clicando no enter. Obs: Para navegar no site em busca do que você quer, depois que pesquisou o tipo do arquivo, basta ficar clicando no enter até achar.

Você Também pode compactar seu PDF em ZIP e usar as seguintes configurações nos parâmetros já citados: 

download="Arquivo zipado" type="application/zip"
Dessa forma o seu navegador Também vai baixar o arquivo do link.
